![app preview](./public/assets/screenshot-readme.png)

### [Live Preview: trelllo.vercel.app](https://trelllo.vercel.app/)

### **Used in this project:**

- Vite
- React Js
- Context API
- Typescript
- React-bootstrap
- React-dnd

### **App Features**

- Add/Remove Lists
- Add/Remove Cards
- Mark Cards as Complete
- Reorder Cards ( Drag & Drop )
