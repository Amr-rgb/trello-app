# Trello App

> a Trello app with new and modern design using React and Typescript

<br/><br/>

### **Used in this project:**

- Vite
- React Js
- Context API
- Typescript
- React-bootstrap
- React-dnd

<br/>

### **App Features**

- Add/Remove Lists
- Add/Remove Cards
- Mark Cards as Complete
- Reorder Cards ( Drag & Drop )
